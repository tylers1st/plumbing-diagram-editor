# plumbing-diagram-editor
I am not a professional programmer, nor very experienced. This is a work in progress. 
I am heavily using github copilot to write this and making occasional changes myself.

This is also a project I'm using to teach myself new things. 

An open-source drag-and-drop tool for creating plumbing training diagrams.

Features (planned):
- Grid-based layout
- Drag-and-drop plumbing parts
- Metadata for pipe size and fittings
- Export to PNG/PDF

Designed for plumbing education and practice drawings.
